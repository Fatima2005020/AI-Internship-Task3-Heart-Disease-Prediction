# Heart Disease Prediction

## Objective
To predict whether a patient has heart disease using machine learning.

## Dataset
Heart Cleveland Dataset

## Steps Performed
- Data loading and exploration
- Data preprocessing
- Model training using Logistic Regression
- Model evaluation (accuracy, confusion matrix, ROC curve)

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Result
The model predicts heart disease with good accuracy.
